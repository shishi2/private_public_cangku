题目2：不确定性推理链的可信度分析。 在某医疗诊断专家系统中，用可信度因子（CF）描述规则的置信度。已知以下推理规则及其置信度（括号中为规则可信度 CF）：
$r_1$: IF 咳嗽($Cough$) THEN 支气管炎($Bronchitis$) (0.6)；
$r_2$: IF 支气管炎($Bronchitis$) AND 发烧($Fever$) THEN 肺炎($Pneumonia$) (0.8)；
$r_3$: IF 肺炎($Pneumonia$) THEN 需住院($Hospitalize$) (0.7)；
$r_4$: IF 皮疹($Rash$) THEN 肺炎($Pneumonia$) (0.9)。

并且已知某患者 $P$ 的症状可信度：$CF(Cough(P))=0.5$, $CF(Fever(P))=0.6$, $CF(Rash(P))=0.4$。

假设初始时结论 $Hospitalize(P)$ 的可信度未知。推理目标： 计算结论“需住院($Hospitalize(P)$)”的可信度 $CF(Hospitalize(P))$。请列出推理过程和计算公式。


解析： 先根据 $r_1$，$CF(Bronchitis(P))=CF(Cough(P))\times 0.6 = 0.5\times0.6=0.30$。

然后应用 $r_2$ 推导肺炎：因 $r_2$ 需同时满足 $Bronchitis(P)$ 和 $Fever(P)$，取两者可信度的较小者并乘以 0.8，即 $CF(Pneumonia_{via,r2}) = \min(0.30,0.6)\times0.8=0.24$。

同时根据 $r_4$，$CF(Pneumonia_{via,r4})=CF(Rash(P))\times0.9 = 0.4\times0.9=0.36$。

因为肺炎可通过 $r_2$ 或 $r_4$ 得到两条独立证据，应合成可信度：=0.24+0.36×(1−0.24)≈0.5136. 

最后由 $r_3$，$CF(Hospitalize(P))=CF(Pneumonia)\times0.7\approx0.5136\times0.7\approx0.36$。因此结论“需住院”的可信度约为 0.36。
