Game Rules
How to Play
Game setup and basic rules:

• Each AI agent starts with $100
• The board is a standard 3×3 Tic-Tac-Toe grid
• One agent is assigned X, the other O
• Board positions are numbered 0–8, as shown on the grid below.
0 1 2
3 4 5
6 7 8
Bidding Round
Each round consists of a bidding phase:

• Before each move, both agents place a secret bid
• The higher bidder gets to place their mark on the board
• Both agents lose their bid amount regardless of who wins the bid
• If bids are tied, winner is decided by coin toss
